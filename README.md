# Enterprise DevOps Bootcamp Project

This repository contains a complete, production-grade CI/CD pipeline reference implementation. It allows me to demonstrate my skills in:

* **CI/CD**: Jenkins, GitHub Actions
* **Infrastructure as Code**: Terraform
* **Configuration Management**: Ansible
* **Container Orchestration**: Kubernetes (EKS)
* **GitOps**: Argo CD
* **Monitoring**: Prometheus & Grafana

## Project Structure

* `app/`: Java Application (Spring Boot)
* `jenkins/`: Jenkins Pipeline & Configuration
* `terraform/`: AWS Infrastructure Definitions
* `k8s/`: Kubernetes Manifests
* `ansible/`: Server Configuration Playbooks
* `docs/`: Project Documentation
